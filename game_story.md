# 《寄宿学校的秘密》详细剧情设计

## 背景设定
**学校**：明德寄宿制中学
**时间**：2024年秋季学期
**主角**：高二学生李明（玩家扮演）
**失踪者**：高三学长张昊，学生会副主席，品学兼优

## 详细剧情线

### 第一章：异常开始
**时间**：周一早上
**事件**：
- 学校突然宣布停课，所有学生必须在宿舍等待通知
- 张昊学长从周日晚上开始失踪
- 学校声称张昊因家庭原因请假回家
- 但李明发现张昊的手机还留在宿舍抽屉里

### 第二章：第一个线索
**发现**：张昊手机中的神秘短信
- 来自"实验组"的4条短信，每条包含一个数字
- 短信内容看似正常，但数字组合成7158
- 指向715宿舍的8号柜子

### 第三章：照片的秘密
**发现**：两张看似相同的合影
- 张昊与李明在图书馆前的合影
- 两张照片拍摄时间相差3天
- 通过日期差计算得出密码：324（3月24日）
- 用于打开715宿舍的密码锁

### 第四章：深入调查
**发现**：715宿舍8号柜子中的线索
- 发现神秘纸条：7158 / 324 / B-7 / ALPHA
- 发现加密U盘，里面有重要文件
- 发现录音带，记录着实验对话
- 发现地下室平面图，标记了B-7房间

### 第五章：真相大白
**发现**：学校的秘密实验和阴暗真相
- 网站揭露学校在进行"优秀学生强化实验"
- 实验涉及对学生进行违规测试
- 录音带揭露张昊发现了实验真相
- 张昊试图举报，被学校控制

## 修改后的结局

### 真相揭露
张昊学长无意中发现了学校领导的严重违规行为：
- 学校领导利用职权进行不法活动
- 张昊收集证据准备举报
- 学校发现后将他控制起来
- 对外谎称"因家庭原因请假回家"

### 最终真相
根据录音带和机密文件：
- 张昊发现了校长的严重违规行为
- 他试图收集证据并向教育部门举报
- 2024年3月24日被控制，现关押在实验楼的秘密房间B-7
- 学校对外宣称"因家庭原因请假回家"

### 游戏结局
玩家收集足够证据后：
- 揭露了学校的秘密实验
- 发现了张昊的下落
- 需要将证据交给有关部门
- 救出张昊学长，曝光学校阴谋

## 详细线索设计

### 线索1：手机短信密码
**来源**：张昊手机
**内容**：
1. "今天的实验数据很完整7" (数字7)
2. "记得带好实验笔记1" (数字1)
3. "晚上记得来实验室5" (数字5)
4. "数据需要重新整理8" (数字8)
**组合**：7158 → 715宿舍8号柜子

### 线索2：照片日期密码
**来源**：两张合影照片
**照片A**：2024-03-21 14:30
**照片B**：2024-03-24 16:45
**计算**：24-21=3，日期差3天
**密码**：324（3月24日）

### 线索3：神秘纸条
**来源**：书桌抽屉
**内容**：手写纸条"7158 / 324 / B-7 / ALPHA"
**用途**：提供密码和房间信息

### 线索4：加密文件
**来源**：U盘
**内容**：多个加密文件
**密码**：需要结合其他线索

### 线索5：录音带
**来源**：隐藏文件夹
**内容**：实验录音
**信息**：透露实验真相和张昊的发现

### 线索6：地下室平面图
**来源**：画的后面
**内容**：实验楼地下室布局
**标记**：B-7房间有特殊标记

### 线索7：实验记录本
**来源**：书桌
**内容**：实验数据和观察记录
**信息**：透露实验方法和副作用

### 线索8：门禁卡
**来源**：书桌抽屉
**用途**：进入实验楼
**限制**：需要特定时间使用

### 线索9：监控录像
**来源**：保安室
**内容**：张昊被带入实验楼的录像
**时间**：需要结合其他线索确定

### 线索10：神秘链接
**来源**：715宿舍发现的手机
**内容**：链接到hiddentruth.mingde.edu
**发现**：网站上揭露了学校的秘密实验

## 游戏流程
1. 玩家进入张昊宿舍，发现手机
2. 从手机短信中发现密码7158
3. 查看照片，发现日期差异
4. 使用密码打开715宿舍8号柜子
5. 发现神秘纸条、加密文件、录音带、平面图
6. 访问隐藏网站，获得更多信息
7. 组合所有线索，揭露真相
8. 完成调查，救出张昊学长

## 无提示设计
- 所有线索需要玩家自行发现
- 不显示"重要发现"等提示
- 密码错误只显示"密码错误"
- 玩家需要自行记录和整理线索
- 增加游戏难度和挑战性
