# 寄宿学校的秘密 - 网页解谜游戏

## 游戏简介

《寄宿学校的秘密》是一个沉浸式的网页解谜游戏，玩家扮演明德中学的学生李明，调查学长张昊神秘失踪的真相。游戏包含多个线索、谜题和互动元素，预计游戏时长约60分钟。

## 游戏特色

- **沉浸式剧情**：完整的故事线和角色设定
- **多样化解谜**：包含数字密码、照片对比、线索搜集等多种解谜方式
- **丰富的视觉效果**：使用现代Web技术打造精美的游戏界面
- **互动式体验**：点击、输入、拖拽等多种交互方式
- **进度跟踪**：实时显示调查进度和已发现的线索

## 游戏目标

通过收集线索、解开谜题，最终揭露学校秘密实验的真相，并找到失踪的张昊学长。

## 主要线索

1. **神秘短信** - 学长手机中的4条短信隐藏着重要密码
2. **照片对比** - 两张照片的日期差异揭示关键数字
3. **密码锁** - 715宿舍8号柜子的密码锁
4. **秘密网站** - 隐藏的网站链接揭露学校机密
5. **实验真相** - 学校的"优秀学生强化计划"

## 如何开始

1. 打开 `index.html` 开始游戏
2. 点击"开始调查"进入游戏主界面
3. 按照提示逐步收集线索和解谜
4. 完成所有线索收集，揭开真相

## 技术实现

- **前端技术**：HTML5 + CSS3 + JavaScript
- **样式框架**：Tailwind CSS
- **视觉效果**：Vanta.js 背景动画
- **图标库**：Font Awesome
- **字体服务**：Google Fonts

## 游戏提示

- 仔细观察每个线索的细节
- 将不同的线索组合起来思考
- 注意数字和日期的含义
- 不要害怕尝试不同的密码组合
- 所有线索都是解谜的关键

## 注意事项

- 游戏包含闪烁效果，对光敏感的请谨慎游玩
