# 《寄宿学校的秘密》游戏架构设计

## 游戏概述
玩家扮演寄宿学校学生李明，调查学长张昊失踪事件。学校突然停课但不放学生回家，要求学生统一在宿舍等待通知。玩家需要通过收集线索、解谜来揭开真相。游戏强调玩家自主探索，不提供任何提示。

## 核心剧情线
1. **第一阶段：发现异常**
   - 学长张昊失踪
   - 学校突然停课
   - 发现学长的手机留在宿舍

2. **第二阶段：线索搜集**
   - 检查学长的手机信息
   - 对比照片发现隐藏信息
   - 解开宿舍密码锁

3. **第三阶段：深入调查**
   - 访问秘密网站
   - 发现学校实验项目
   - 揭开失踪真相

## 线索系统设计（无提示版本）

### 线索1：手机短信密码
**来源**：张昊手机
**内容**：
1. "今天的实验数据很完整7"
2. "记得带好实验笔记1"
3. "晚上记得来实验室5"
4. "数据需要重新整理8"
**组合**：7158 → 715宿舍8号柜子
**难度**：需要玩家自行发现数字规律

### 线索2：照片隐藏信息
**来源**：两张合影照片
**照片A**：2024-03-21 14:30
**照片B**：2024-03-24 16:45
**隐藏信息**：日期差3天，时间差2小时15分
**密码**：324（3月24日）或215（2小时15分）
**难度**：需要玩家计算日期和时间差

### 线索3：神秘纸条
**来源**：书桌抽屉
**内容**：手写纸条"7158 / 324 / B-7 / ALPHA"
**用途**：提供密码和房间信息
**难度**：需要玩家理解纸条含义

### 线索4：加密文件
**来源**：U盘
**内容**：多个加密文件
**密码**：需要结合其他线索
**难度**：需要玩家尝试不同密码组合

### 线索5：录音带
**来源**：隐藏文件夹
**内容**：实验录音
**信息**：透露实验真相
**难度**：需要玩家找到播放设备

### 线索6：地下室平面图
**来源**：画的后面
**内容**：实验楼地下室布局
**标记**：B-7房间有特殊标记
**难度**：需要玩家理解地图符号

### 线索7：实验记录本
**来源**：书桌
**内容**：实验数据和观察记录
**信息**：透露实验方法和副作用
**难度**：需要玩家理解科学术语

### 线索8：门禁卡
**来源**：书桌抽屉
**用途**：进入实验楼
**限制**：需要特定时间使用
**难度**：需要玩家发现使用时间

### 线索9：监控录像
**来源**：保安室
**内容**：张昊被带入实验楼的录像
**时间**：需要结合其他线索确定
**难度**：需要玩家找到正确时间段

### 线索10：神秘网站
**来源**：715宿舍8号柜子中的手机
**内容**：hiddentruth.mingde.edu
**信息**：学校机密档案
**难度**：需要玩家自行探索网站内容

## 新增线索类型

### 音频线索
- 录音带中的实验对话
- 需要玩家仔细聆听内容
- 可能包含密码或重要信息

### 地图线索
- 地下室平面图
- 需要玩家理解空间关系
- 标记重要地点和路径

### 加密文件
- 需要密码才能打开
- 密码分散在其他线索中
- 增加解谜复杂度

### 时间线索
- 监控录像的时间戳
- 需要玩家计算时间差
- 结合多个时间点推理

## 游戏结局
**真相**：学校在进行违规的教育实验，张昊发现了真相后被控制
**救援**：玩家收集足够证据后，可以救出张昊
**结局**：学校实验被曝光，相关责任人被处理

## 难度设计
- **初级**：手机短信数字规律
- **中级**：照片日期时间计算
- **高级**：多线索组合解谜
- **专家**：加密文件破解

## 无提示设计原则
1. 不显示"重要发现"等提示
2. 不自动高亮关键信息
3. 不提供解谜思路
4. 所有线索需要玩家主动发现
5. 密码错误只显示"密码错误"
6. 玩家需要自行记录和整理线索
